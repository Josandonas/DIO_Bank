# DIO Bank 

> ### Ações do Banco:
>
>> * Sacar
>> * Tranferir
>> * Depositar
>> * Listar Contas
>> * Inserir Nova Conta

> ***Os menus estão na pasta View***
> 
> ***Os métodos estão na pasta Controller***
> 
> ***Pasta Model não existe, porque não se está usando um banco de dados***

### Ambiente

Para funcionar localmente, precisa ter instalado:
* .NET 3.2
* VisualCode 1.55.2

### Build

```
dotnet build
```

### Execução

```
dotnet run
```
