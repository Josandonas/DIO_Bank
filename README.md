# DIO Bank 

> No presente projeto são usados os métodos:

>> * Sacar
>> * Tranferir
>> * Depositar
>> * Listar Contas
>> * Inserir Nova Conta

### Ambiente

Para funcionar localmente, precisa ter instalado:
* .NET 3.2
* VisualCode 1.55.2

### Build

```
dotnet build
```

### Execução

```
dotnet run
```
